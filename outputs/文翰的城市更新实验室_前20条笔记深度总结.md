### 2. 阿云朵仓，在贵州2年把4000方做到1个亿（4676赞）⭐重点

**类型**：视频 · 商业案例深度拆解

**核心内容**：
这是文翰最受关注的商业案例笔记，详细拆解了贵阳阿云朵仓从一个闲置底商/烂尾项目翻盘为年营收1亿的标杆非标商业的全过程。

**视频要点**：

**① 项目背景**：
- 前身：大唐东原财富广场（闲置底商/烂尾项目）
- 商业面积：4000㎡（仅已开业部分）
- 2025年营收：1亿元+，客流500万人次+
- 2026年Q1：营收和客流同比翻倍

**② 操盘方反直觉逻辑（三招翻身）**：
- **第一招：先让人待住，再谈招商**
  - 60%面积做公共空间，仅30%规划商铺
  - 打造洗马书苑、银杏礼堂、丘浪滑板场、戏台等免费开放公共场景
  - 拆除所有外墙广告牌、灯箱、招牌，让场地"放松下来"
  - 核心理念：**这里不是来消费的，是来把时间放下来的**

- **第二招：拒绝标准化连锁，招愿意扎根的主理人**
  - 引入100+主理人品牌，无一连锁品牌
  - 业态聚焦：精品餐饮、独立咖啡、非遗手工、古着、书屋、精酿等
  - 合作模式：联营分成（餐饮扣点10%-30%，T+1分账）
  - 选店标准：老板自己喜不喜欢、主理人愿不愿意长期扎根

- **第三招：用活动填满日历对抗一次性打卡**
  - 2025年做了500多场活动（面包节、精酿节、滑板赛、脱口秀、市集等）
  - 核心逻辑：打卡是一次性的，活动才是复购

**③ 设计理念**：
- 名称来源：贵州彝语"我们的理想之地"
- 室内概念："洞穴中的村落"（由几里建筑齐帆设计）
- 四大主题场景：村口市集、长角食集、余庆粮仓、十字街
- 三座文化地标：银杏礼堂、洗马书苑、红砖剧场

**④ 团队与运营**：
- 专注商业的核心人员约2人，品牌团队7-8人，其他兼职
- 用人标准：热爱>学历，要有见识和认知
- 物业属性：开发商全自持，目的是激活酒店（珍珀酒店）和物业
- 收入来源：底金+流水抽成联营

**核心观点**：
这个项目最特别的是**操盘方原本没做过商业**，从酒店配套出发，用"反商业"的逻辑做出了高坪效的非标商业样本。

**涉及项目**：阿云朵仓、曹国梁（贵州视界集团）、几里建筑

### 3. 停业20年的友谊商店翻红，有什么高人指点（395赞）⭐重点

**类型**：视频 · 城市更新案例

**核心内容**：
深度拆解北京友谊商店从停业20多年的"沉睡遗产"翻红为北京最火的非标街区的全过程。

**视频要点**：

**① 项目背景**：
- 成立于1964年，1972年迁至建国门现址，是国家重点外事工程
- 80-90年代凭外汇券可购买进口商品，是最神秘的商场之一
- 90年代后商业逐步开放，该店因历史遗留问题停滞约20多年
- 2021年并入王府井集团后重新启动转型改造

**② 五大核心策略**：
1. **开放空间**：拆除围墙，将封闭大院变为800平米中央草坪+环形步道
2. **轻改氛围**：保留建筑原貌（维修车间、冷库、传达室等），轻改造让复古与流行碰撞
3. **品牌组合**：引入北平机器、超级转转、铁手咖啡等特色概念店
4. **情绪营造**：定期热点活动、节日氛围、潮牌快闪
5. **轻资产运营**：5000万低投入，给商家低租金和长租期

**③ 经营数据**：
- 占地18,000㎡，建筑面积近25,000㎡（一期约2万㎡）
- 目前已满租，平日客流五六千，周末达1.2-1.3万
- 开业半年31家店做2亿多营收，2026年预计超5亿元
- 投资当年回本，业绩同比翻12倍

**核心观点**：
友谊商店的改造启示在于**"不刻意的未完成感"**——让商业自然生长，而不是过度设计。

**涉及人物/项目**：刘杰（王府井集团北京友谊商店总经理）、屈太浪（城市观察博主）

### 4. 非标商业，正在进入淘汰赛。（160赞）

**类型**：视频 · 行业趋势判断

**核心内容**：
纪文翰提出"非标商业淘汰赛"概念，分析了当前非标商业赛道正在经历的洗牌周期。

**视频要点**：
- **定义**：什么是"非标商业"——区别于标准化购物中心，主打特色、设计感、主理人品牌的商业体
- **淘汰原因**：
  1. 大量项目跟风模仿，同质化严重
  2. 缺乏长期运营能力，"重设计轻运营"
  3. 流量红利消退，打卡经济见顶
  4. 资本退潮，融资变难
- **存活法则**：
  1. 有独特的内容生产能力（如阿云朵仓的活动运营）
  2. 主理人生态建设（不是简单招商）
  3. 空间氛围的真实表达，而非刻意"网红化"
  4. 长期主义心态，不追求快速回本

**核心观点**：
非标商业已经从"概念期"进入"实战期"，只有真正有内容、有运营能力的玩家才能活下去。

## 核心洞察总结

### 1. 非标商业的"三不"原则
- **不迷信大拆大建**：用设计拉情绪，用运营留人流
- **不追求标准化**：主理人生态比连锁品牌更重要
- **不打一次性牌**：用活动填充日历，让复购成为可能

### 2. 存量商业转型的底层逻辑
- 从"货物收割机"到"时间容器"：商场不再是卖货的地方，而是让人停留的空间
- 从"收租模式"到"联营分成"：利益绑定才能长期共赢
- 从"封闭盒子"到"城市连接器"：打破建筑边界，承接公共空间

### 3. 城市更新的人文维度
- 人的记忆比建筑更重要（一丝厂老员工的故事）
- 文化名人是城市更新的精神引擎（许知远、青山周平）
- 留白是最高级的设计（马头巷的"不产生营收的休息空间"）

### 4. 主理人生态的运营方法论
- 选店标准：老板自己喜不喜欢 > 商业可行性
- 合作模式：联营分成 > 固定租金
- 用活动对抗打卡：500场/年是阿云朵仓的复购密码
